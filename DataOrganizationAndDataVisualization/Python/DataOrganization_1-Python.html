#LAMBDA  uma funo annima, criada em tempo de execuo.
# aplicada uma funo (LAMBDA) a uma lista de elementos, agregando o resultado at que o ltimo elemento seja processado.
#o retorno  justamente a agregao de todas as operaes executadas pela funo em questo, neste caso, a LAMBDA.

sub = reduce((lambda i, j: i-j), array4)

print('operao com base na funo REDUCE sobre o array informado -> 1-2 = -1 - 3 = -4 - (-20) = +16', '\n')
print('resultado do REDUCE sobre o array informado:', sub)
